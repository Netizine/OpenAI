# Security Policy

## Supported Versions

The following versions of the OpenAI project are
currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| .NET Framework 4.6.1+   | ✅: |
| .NET Standard 2.0+   | ☑️ :                |
| .NET Core 2.0+   | ✔️: |
